# WanderNest Journeys — Travel Site

This is a single-page marketing and itinerary site for a boutique travel company called **WanderNest Journeys**.

### Features

- **Trip gallery**: See different curated trips with duration, theme, and quick highlights.
- **Itinerary modal**: Click any trip to open a detailed, day-by-day itinerary in a sleek modal.
- **Download itineraries**: From the modal, download a ready-to-share `.txt` file version of the itinerary.
- **Responsive & modern UI**: Dark, glassmorphism-inspired design that looks good on desktop and mobile.

### Running the site

You don’t need any build tools:

1. Open the project folder: `c:\Users\Soma\Desktop\Projects\Travell`
2. Double-click `index.html` to open it in your browser  
   **or**  
   right-click `index.html` → “Open with” → your preferred browser.

Everything (HTML, CSS, JS) is loaded locally with no external dependencies besides Google Fonts.
